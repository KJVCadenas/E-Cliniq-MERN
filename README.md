# E-Cliniq: Clinic Management System

E-Cliniq is a web-based clinic management system designed to help nurses manage essential clinic processes such as maintaining patient records, scheduling appointments, monitoring medicine inventory, and generating monthly reports. It serves as a digital tool to improve the speed and organization of day-to-day clinic operations.

I chose to rebuild this project using the MERN stack as a way to measure how much I have improved since first developing it as a capstone project in my undergraduate studies. Revisiting E-Cliniq allows me to apply more modern tools, better practices, and cleaner code architecture to a project I was once proud of as a student.

## Features

- Create and manage digital health records
- Set, update, and cancel patient appointments with reminders
- Track and manage medicine inventory with low-stock alerts
- Generate digital reports for monthly accomplishments and inventory summaries

## Tech Stack

### Frontend
- Vite + React
- React Router v6
- Tailwind CSS or ShadCN/UI
- Axios
- React Hook Form and Zod for form handling and validation

### Backend
- Node.js + Express.js
- MongoDB + Mongoose
- JWT and bcrypt for authentication
- Optional: Nodemailer for email notifications, PDFKit or Puppeteer for PDF report generation

### Deployment (Future)
- Frontend: Vercel or Netlify
- Backend: Render or Railway
- Database: MongoDB Atlas
